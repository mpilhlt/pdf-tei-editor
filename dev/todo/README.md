# Code Assistant Tasks

This directory contains tasks for the code assistant comprising

- bug fixes 
- new features
- feature enhancements
- code reorganization
- documentation improvements
- etc.

For each file, create an issue on GitHub using the `gh` tool. In the issue, summarize the task and reference the file with its static github URL.

While working on the task, add summaries of what you have implemented to the file. 

After you are done with each task, move it to `../done` and in the commit message, statically reference the file in its new location.
